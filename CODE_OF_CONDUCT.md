# Code of Conduct

We want NotepadSharp to be a welcoming, harassment-free project.

## Be respectful
- Be kind and constructive
- Assume good intent
- Focus on the technical discussion, not the person

## No harassment
Harassment and discriminatory behavior are not tolerated.

## Enforcement
Project maintainers may remove content or contributions that violate this code of conduct.

## Reporting
If something happens, contact the maintainers (via GitHub or the channel listed in `SECURITY.md`).
